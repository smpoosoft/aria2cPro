# 开发需求

## v1.0.0
写一个单文件 html 文件，界面上通过输入框，下拉，数字输入等原生 html 组件，输入 aria2c 参数，然后最下面一行，用 div 显示最终形成的 aria2c 命令。
div 行提供点击复制到剪贴板的能力。
使用 html5 + css3 + <script type="model"> 模式。单文件，不依赖任何第三方

## v1.0.1
1. 模型下载 URL 输入框，不要预设任何默认值。然后点击时，如果输入框值为空，可以自动将剪贴板内容粘贴进去。支持右侧清空按钮。
2. 保存目录 (-d) 输入框，应该是一个左右结构的输入框，右边是一个按钮，点击按钮，可以打开弹窗，选择本地文件夹。
3. 磁盘预分配 (--file-allocation) 独占一行，每个下拉选项，后面需要提供简单描述，如果描述文本过长，那么就把 命令编码和描述文本上下排列，描述文本允许换行，不要求必须一行显示完整。主要以内容为准

## v1.0.3
1. 保存文件名 (-o) 不要任何默认值。
2. 当 模型下载 URL 有值以后（无论是输入还是粘贴），自动解析下载地址的文件名作为 保存文件名  输入框的默认值。
3. 页面加载时，自动创建 indexedDb ，名为 localAria2c，里面提供一个下载历史记录表，然后页面分为左右结构，左侧是下载历史，右侧是当前修改之前的页面内容。
4. 让页面再精美一点
